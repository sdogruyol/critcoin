# critcoin

Crystal ve Kemal ile 100 satırdan az kod ile kendi blok zincirinizi geliştirin.

## Kurulum

```
git clone https://github.com/sdogruyol/critcoin
cd critcoin
shards install
```

## Çalıştır

```
crystal src/critcoin.cr
```
